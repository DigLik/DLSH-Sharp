# Встроенные команды DLSH

Встроенные команды исполняются непосредственно процессом оболочки, что обеспечивает высокую скорость работы.

## `print`
Выводит текст или значения переменных в консоль.
**Синтаксис:** `print [аргументы...]`
**Пример:** `print Текущий путь: $PWD`

## `cd`
Изменяет текущую рабочую директорию.
**Синтаксис:** `cd [путь]`
- Без аргументов переходит в домашнюю директорию (`$HOME`).

## `var`
Универсальная команда для работы с окружением и вычислениями.

### Просмотр и установка
- `var` — список всех переменных.
- `var NAME=VALUE` — установка значения.
- `var del NAME` — удаление переменной.

### Математические вычисления (`var match`)
DLSH использует Roslyn для вычислений, поэтому поддерживаются любые корректные выражения C#.
**Синтаксис:** `var match [имя] = [выражение]`
**Пример:** `var match x = Math.Sqrt(144) + 10`

## `alias`
Управление сокращениями команд.
- `alias list` — показать все алиасы.
- `alias add name=command` — добавить новый.
- `alias del name` — удалить.

## `clr`
Очищает экран терминала.

## `exit`
Завершает работу оболочки. Можно передать код выхода: `exit 1`.
